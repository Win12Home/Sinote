## 简单介绍
Sinote是一个基于PySide6的***开源*** 编辑器。<br>
这个编辑器允许你创作符合规范的**自定义插件**！如需查看文档：[这里就有](../documentation/pluginEdit/plugin.md)
<br>Sinote在英文的意思为：**SI**mple **N**ative P**O**r**T**able **E**ditor，也就是**轻量的原生编辑器**<!--我说什么呢！-->。
<br>我们推荐你用这些系统来运行：
 - Windows **Vista**以上（可能不能运行，Win10一下可能需要安装VC Runtime）
 - Linux（运行**可执行文件**即可）
 - macOS（没试过，但编译成功了）
<!--天无雨地焦旱~鬼子眼睛皆发蓝~-->
你真想用也不是没有办法，在命令行用``--bypass-system-check``即可。

## 怎么安装
<!--谁都会吧-->
在Windows、Linux、macOS上，安装**极其的**简单！
<br>你只需要打开**Release Page**，然后找到最新的版本即可！<br>
<br>
反观其他系统，他们就不太好弄。
<br>你需要克隆（不是fork）它然后再用Python运行。
```shell
git clone https://github.com/Win12Home/Sinote.git
cd Sinote/
pip install -r requirements.txt -i https://mirrors.tuna.tsinghua.edu.cn/pypi

python SinoteMain.py --bypass-system-check
```
